10869 : 사칙연산

두 자연수 A와 B가 주어진다. 이때, A+B, A-B, A*B, A/B(몫), A%B(나머지)를 출력하는 프로그램을 작성하시오.

예제입력
7 3
예제출력
10
4
21
2
1

내가 쓴 정답

const fs = require('fs'); const numbers = fs.readFileSync('/dev/stdin').toString().split(' '); const A = Number(numbers[0]); const B = Number(numbers[1]); console.log(A+B); console.log(A-B); console.log(A*B); console.log(parseInt(A/B)); console.log(A%B);

다른 사람 정답 정리

  • parseInt()를 Math.floor()로 작성한 경우가 있음

2588 : 곱셈

(세 자리 수) × (세 자리 수) 차례대로 출력하기

예제입력
472
385
예제출력
2360
3776
1416
181720

내가 쓴 정답

const fs = require('fs'); const numbers = fs.readFileSync('/dev/stdin').toString().split('\n'); const num1 = Number(numbers[0]); const num2 = numbers[1].split(''); const results = num2.map(num => Number(num) * num1); console.log(results[2]); console.log(results[1]); console.log(results[0]); console.log(numbers[0] * numbers[1]);

다른 사람 정답 정리

  • parseInt()를 Math.floor()로 작성한 경우가 있음
